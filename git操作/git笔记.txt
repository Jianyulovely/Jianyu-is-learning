1. 更新远程分支信息（这一步不会改代码，只是让 Git 知道远程多了哪些分支）
git fetch 

2. 查看远程分支
git branch -a

3. 切换到某个分支
git checkout -b branch_name branch_name origin/branch_name

意思是：
创建本地分支 branch_name
并跟踪远程 origin/branch_name

4. 查看分支
git branch

5. 合并分支到主分支
确认在哪个分支：
git branch

合并时候一定要在目标分支上
git checkout main

更新主分支，防止不是最新
git pull origin main

合并另一个分支
git merge dev

6. 查看当前分支结构
git log --oneline --graph --all --decorate
